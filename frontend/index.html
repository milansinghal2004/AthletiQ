Core Capabilities

What AthletiQ Does

🦴
Pose Estimation
Extracts 33 skeletal keypoints per frame using MediaPipe Pose, with confidence scoring and temporal smoothing.
MediaPipe · Real-time
🏏
Shot Detection
Classifies batting shots — drive, pull, sweep, cut and more — from skeletal trajectory and joint velocity signatures.
ML Classification · 8 Shots
📐
DTW Analysis
Dynamic Time Warping aligns your motion with expert reference clips, handling tempo variation for true similarity scores.
DTW · Joint-level
⚡
Clip Extraction
Locates batting action windows within full footage, isolating each stroke for frame-accurate analysis automatically.
OpenCV · Auto-trim
🎯
Performance Scoring
Per-joint and aggregate technique scores reveal exactly which body segments deviate from optimal form.
Quantitative · Per-joint
🤖
AI Feedback
Converts numerical deviations into natural language coaching cues — specific, actionable corrections per joint.
LLM-powered · Actionable
